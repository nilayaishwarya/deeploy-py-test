# Deeploy Python Client

Python client for interacting with Deeploy.

## Documentation

### Generate

1. Generate the markdown from python code

    ```bash
    pydoc-markdown --render-toc -v
    ```

2. Generate the HTML from markdown

    ```bash
    mkdocs build -f docs/mkdocs.yml
    ```
